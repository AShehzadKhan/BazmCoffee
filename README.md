Bazm-Coffee ☕

A clean, modern, and fully responsive café website built using HTML and vanilla CSS.
It features a dark-cyan gradient theme, a sticky navbar, hero banner, about section, menu grid, and a contact/footer area — all optimized for mobile, tablet, and desktop.

Features

Responsive design (no JavaScript)

Sticky navigation bar with hover effects

Hero section with background image

About section with image + text layout

Coffee menu cards with hover effects

Contact section with social icons


Technologies Used

HTML5

CSS3 (Flexbox, Grid, Media Queries)


How to Use

1. Clone or download the project


2. Open index.html in any browser


3. Customize images, text, or colors as needed



License

Free to use and modify.
